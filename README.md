# PersonalDetailsApp

A Mobile Application with Personal Details. 

### Technologies used for this project
- Webkit
- UIKit
  
### Approach Description
##### The UI 
- Programmatically drew up the UI

##### Landing Page
- Displayed my name, slack name and track clearly.
- Displayed my slack image
- Added a button with an action that leads to my github page
  
##### Github Page
- Added a back button for navigation.
- Added a page label that describes what the whole page is about.
- Used webkit to display my github page.

### Setup Instruction
- Clone this repository with this link below.
  ```
    git clone https://github.com/tesddev/PersonalDetailsApp.git
  ```
- Open the app with an Xcode and run the app in a simulator or a physical device.
